This folder contains the uPOWER Assembler and Simulator. It also include two programs 1) Sum of two numbers 2) Sum of an array.

The steps to run the assembler in Windows are:
i) Open the command terminal and type py assembler.py
ii) Once the program runs, a prompt will ask for the filename of the program

The steps to run the simulator in Windows are:
i) Open the command terminal and type g++ simulator.cpp
ii) Then to run the program, type a
