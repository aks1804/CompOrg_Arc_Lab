This folder contains the entire uPOWER datapath along with the Control signals being provided with the help of the Main Control Unit.

The steps to run the datapath are :
i) Modify the registers, instructions and data memory accordingly
ii) In the command terminal, run iverilog uPowerDatapath_tb.v and then vvp a.out
